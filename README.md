# Novacoin (Sepolia) — Crowdfunding dApp + Reward Token

This project is a simple Ethereum crowdfunding dApp deployed on **Sepolia**.
Users can create fundraising campaigns, contribute ETH, and receive **NOVA tokens** as rewards.

## Features
- Connect MetaMask
- Create campaigns (goal + duration)
- Contribute ETH to campaigns
- Automatically mint NOVA reward tokens for contributors
- View campaigns list and progress

## Smart Contracts
- `NovaCoin.sol` — ERC20-like reward token
- `NovaFunding.sol` — Crowdfunding logic + token minting rewards

## Network
- Testnet: **Sepolia**
- Chain ID: `11155111` (`0xaa36a7`)

## Deployed Contract
